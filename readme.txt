Содержимое архива:
parser_manager_version_1 - модуль для версий 1.3.х - 1.6.х
parser_manager_version_2 - модуль для версий 2.1.х - 1.2.х
parser_manager_version_3 - модуль для версий 2.3.х
for all versions - библиотеки парсера

Установка:
1. Скопировать содержимое из каталога parser_manager_version_x  в папку магазина с директорией admin
2. Скопировать содержимое из каталога for all versions(php x)  в папку с директорией system
	php 5 - для версий c php 5
	php 7 - для версий c php 7
